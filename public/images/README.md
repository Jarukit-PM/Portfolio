# Images Directory

This directory contains all static images used in the portfolio website.

## Folder Structure

### `/logos/`
Company logos and brand images used in the experience section or project showcases.

**Usage example:**
```tsx
<Image 
  src="/images/logos/company-name.svg" 
  alt="Company Name"
  width={40}
  height={40}
/>
```

### `/profile/`
Profile pictures and personal photos.

**Usage example:**
```tsx
<Image 
  src="/images/profile/profile-pic.jpg" 
  alt="Profile Picture"
  width={200}
  height={200}
/>
```

## File Formats
- Preferred: SVG (for logos), WebP or JPG (for photos)
- Ensure images are optimized for web use
- Keep file sizes reasonable for fast loading
